# GENERATIVE-AI-PROJECTS-1
Project 4: AI-Based Timetable Generator Using Python
Project Overview
This project implements an automated timetable generation system that dynamically creates academic schedules based on user-provided subjects, working days, and number of periods.
Objective
To understand structured scheduling logic, data handling, and automated CSV export functionality using Python.
Technologies and Tools
	Python
	Pandas
	Random Module
Key Learning Outcomes
	User input validation
	Randomized scheduling logic
	DataFrame creation and manipulation
	Exporting structured data to CSV
Applications
	School and college scheduling
	Academic planning tools
	Routine generation systems
